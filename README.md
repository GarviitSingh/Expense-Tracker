# Minimalist Expense Tracker

A sleek, dark-themed expense manager built with Vanilla JavaScript. This project focuses on clean UI/UX and efficient data handling using the browser's LocalStorage.

##  Features
- **Minimalist Dark UI:** Designed with a deep charcoal palette for reduced eye strain.
- **Full CRUD:** Create, Read, Update, and Delete transactions seamlessly.
- **Smart Math:** Dynamic calculation of Total Balance, Income, and Expenses using Array methods.
- **Indian Currency Formatting:** Implements `en-IN` locale (₹) for proper comma placement (Lakhs/Crores).
- **Data Persistence:** Your data stays saved even after refreshing the page.

##  Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Custom properties (variables) and Flexbox/Grid.
- **JavaScript:** Vanilla JS (ES6+), LocalStorage API, and `Intl.NumberFormat`.
